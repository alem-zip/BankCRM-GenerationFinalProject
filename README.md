# ProjectWorkSFDITA021
Questa repository contiene i metadati del progetto Salesforce . Per
iniziare a lavorare sulla tua Scratch Org locale , segui questi
passaggi :
## 1. Prerequisiti
- ** Visual Studio Code ** e ** Salesforce CLI ** installati .
- Una Org Salesforce Developer con il ** Dev Hub ** abilitato ( Setup
-> Dev Hub -> Enable ).
## 2. Autorizza il tuo Dev Hub
Apri il terminale all ’ interno di questa cartella e collega la tua
Org personale eseguendo :
‘sf org login web -d -a MioDevHub ‘
## 3. Lancia il setup automatico
Per creare la Scratch Org ( chiamata * ProjectWork *) , caricare i
metadati e assegnarti i permessi corretti , lancia lo script
preparato :
- ** Se usi Mac / Linux :** Esegui da terminale ‘./ setup_ambiente .sh ‘
- ** Se usi Windows :** Fai doppio clic sul file ‘ setup_ambiente . bat ‘
( oppure lancialo da terminale )
Al termine dello script , la Org si aprira ’ automaticamente nel tuo
browser , pronta per essere utilizzata!
